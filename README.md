## Mac 时间追踪软件是什么，Qbserve 又是什么

**Mac 时间追踪软件**，指的是运行在后台、自动监测你的 Mac 使用行为并生成效率报告的工具。它不同于番茄钟或手动打卡类工具——核心区别在于「自动」两个字。你不需要记得开始，不需要记得停止，它就在那里，安静地帮你记录一切。

Qbserve 是这类工具里的代表产品，由独立开发团队 QotoQot 开发，专为 macOS 设计。它内置了一个包含 8100+ 个应用、网站和游戏的本地数据库，能自动判断每项活动属于「高效」「中立」还是「分心」三类，并以图表形式呈现你每天、每周、每月的生产力趋势。

所有数据只存在你自己的电脑上，没有云同步，没有账号，没有数据上传。

## Qbserve 能帮你看到什么

### 把时间分成三种颜色

打开 Qbserve，你会看到一张饼图把今天的时间切成三块：高效（绿色）、中立（灰色）、分心（橙红）。

Slack 的工作频道和 B 站，在 Qbserve 眼里是两种完全不同的事情。YouTube 上的技术教程和综艺节目，也能被区分开来。你还可以手动调整分类，比如把知乎标记为「高效」（如果你真的在上面学习的话）。

### 菜单栏里的实时百分比

Qbserve 会在 Mac 菜单栏显示一个数字——当前「高效活动」占总使用时间的百分比。抬头看一眼，就知道今天状态怎么样。

数字低了，是一个提醒。Dock 栏图标的颜色也会随着状态变化。这两个小细节，让你不需要特意打开报告，就能感知到时间的去向。

### 按周、月看趋势

你周一效率最高还是周五？上午专注还是下午容易分心？Qbserve 的报告可以按天、周、月切换，让你看出自己真实的工作节律。

自由职业者拿这个来核算项目时间，学生用它发现自己开着课件刷视频的真相，远程工作者用它找回对工作时间的掌控感。

### 项目追踪 + 发票生成

这是 Qbserve 对自由职业者最有用的功能之一。你可以设定规则：只要打开某个文件路径、或者访问带有特定字符串的网址，就自动计入某个项目的工时。收工的时候，一键导出 JSON / CSV，或者直接生成支持 18 种语言的发票。

客户追问"你这个任务花了多少小时"，你不需要凭感觉，数据在那里。
